#Black jack (java)

a simple command line blackjack written game in java.
using standard dealer vs player rules

##features 
- play blackjack against dealera cpmputer dealer
- hit and stand actions
- automatic dealer logic
- win / lose detection
- score calculations with face cards and aces

##how to run 
- make sure u have isntalled java (verision 8 or higher)
- clone this repoository
- navigate to the project folder
- complie the program
- run the game

- ## how to play
- start with two cards
- press 1 for 'hit' to draw another card
- press 2 to 'stand' to end turn
- dealer draws cards until he beats u or he busts
- closest to 21 wins

## project dtructure 
- control.java - starts the game
- deck.java -creates deck and shuffles cards
- cards.java - represents a playing card
- hands.java - handles different hands 
- blackjack.java - backend logic
